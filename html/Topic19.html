Interface Example (Graupner/JR)

This section shows an example of a basic transmitter interface for Graupner/JR transmitters. The set-up uses the complete pulse signal from the receiver. This way you can wirelessly control the simulator with your transmitter.

Note  The interface also works with other simulators like FMS and FlyRC.




Note  The final product.



Note  The grey plug the interface is connected to is an extension cable that is plugged into the parallel port.



Note  With Graupner/JR equipment the complete pulse signal can be found on the pin in the receiver's battery socket next to the two pins that the battery is normally connected to. The orange lead is connected to this pin. To power the receiver, plug the battery into a servo socket.
